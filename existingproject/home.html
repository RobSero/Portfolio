PHILOSOPHY

A lot of my design philosophy originates from working on residential construction projects where the intent was to create spaces that positively impact the residents through clean and clear designs. My aim is to translate these concepts into the designs of my apps and sites in the hope of the same results.


I tend to follow calm technology principles where the tech is a tool for the user to experience a website or service naturally rather than the tech dominating the site and battling for the users attention.


I find this approach to web design allows for more natural interactions between the user and the product or service.

RENTGAP

A full-stack platform for users to sign up and invest within the residential property market with the opportunity to earn capital gains and regular monthly income through rent.


The stack includes React front end alongside a Django back -end supported by a SQL database.

Data is consumed from a third- party API, which is snapshot once a user has invested in a property and this enables the user to track if their investments have fluctuated over time.

INDIANA JONES

Recreating Pac-Man using only HTML, CSS and vanilla JavaScript. Game design and logic was created from scratch with win/lose conditions and score tracking capabilities.

The core feature of the game was creating the ghost tracking logic and their behaviours as their movement is not random and their actions are determined by multiple factors, such as Pac-Man’s location relative to theirs; death timers and whether they are chasing or fleeing.

The game is quite heavily maths- oriented and is largely derived from Pythagoras theorem to calculate relative positions and overlaying many intervals/timeouts.

CORE TECHNOLOGIES

  • HTML5
  • CSS3
  • VANILLA JAVASCRIPT

BLOG

A personal project where I have built a simple platform to make micro-posts with the intent of experimenting with SEO, analytics and digital marketing.

The app’s stack is relatively simple with a django backend to utilise the features of the django admin page to make posts quickly. The client side uses a React app which was intentional as it is an opportunity to explore how SEO works with single page applications.

The research and experiment purpose of the app means that setup and maintaining the blog is simple as there are no requirements for user accounts or validation. Micro-blogs also allow me to find time to put out regular content.

SPOT.ME APP

Team project utilising the MERN stack and third-party APIs to build a platform for users to find local fitness facilities, create groups and post events between followers.

My role in the project was to work on the back-end creating a wide array of controllers, models and RESTful APIs which enabled the team to consume the required data they needed on the front-end.
