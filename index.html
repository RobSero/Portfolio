Hi there, I am Rob and I love building things! With the world increasingly relying on the internet, I believe people will start to care about their digital space just as much as our real world environment. I strive to build things which convey a positive and clear space which works with you and not against.


I have always had an affinity for building things, and programming has opened up so many exhilarating opportunities for me to flex both my creative and technical mindsets.


With a background in architectural technology and startup experience, I have a track record of project management, client handling and technical design of the physical realm, which has been invaluable for my career in tech as a Full-Stack Developer.

Philosophy

A lot of my design philosophy originates from working on residential construction projects where the intent was to create spaces that positively impact the residents through clean and clear designs. My aim is to translate these concepts into the designs of my apps and sites in the hope of the same results.


I tend to follow calm technology principles where the tech is a tool for the user to experience a website or service naturally rather than the tech dominating the site and battling for the users attention.


I find this approach to web design allows for more natural interactions between the user and the product or service.

Indiana Jones Temple Runner

Vanilla JS Pac-Man Game

HTML / CSS / JavaScript

Recreating Pac-Man using only HTML, CSS and vanilla JavaScript. Game design and logic was created from scratch with win/lose conditions and score tracking capabilities.


The core feature of the game was creating the ghost tracking logic and their behaviours as their movement is not random and their actions are determined by multiple factors, such as Pac-Man’s location relative to theirs; death timers and whether they are chasing or fleeing.


The game is quite heavily maths-oriented and is largely derived from Pythagoras theorem to calculate relative positions and overlaying many intervals/timeouts.

eBuddy - eBay Price Checker

Web Scraping App / iOS & Android Native

Angular / Flask / React Native / Excel Spreadsheet Writer

This project was a solo two day sprint to produce an app to help my parents out as they love visiting boot sales and picking up bargains. The app will quickly search and scrape eBay listings to find an average buyout price of an item with a simple interface for a quick result.


The project has two versions: The Angular web app and a React Native app for iOS and Android but both of these interface with a RESTful Python/Flask API. Despite simple in concept and design, the app uses a range of useful technologies to harvest and handle the data such as the ability to write Excel spreadsheets for the user to download.


This project was a chance to test out Angular, Flask and React Native and whilst I can start to appreciate the value in these, particularly Flask for such a quick server side API - I believe React and Django are a far more robust option for most scenarios and will be leaning more towards these technologies in future projects.
